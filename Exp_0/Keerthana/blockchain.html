Blockchain Basics

This webpage is created to explain the basics of blockchain technology in a simple way.

What is Blockchain?

Blockchain is a digital record system where information is stored in blocks and connected together.

How Blockchain Works

  1. A transaction is created
  2. The transaction is verified
  3. The data is stored in a block
  4. The block is added to the chain

Main Features

  • Decentralized system
  • High security
  • Transparent records
  • Data cannot be changed easily
